# 📝 To-Do List Web App

A simple and responsive To-Do List application built using **HTML**, **CSS**, and **JavaScript**. This project allows users to add tasks, mark them as completed, and delete tasks with an intuitive and modern user interface.

## 🚀 Features

* Add new tasks instantly.
* Mark tasks as completed.
* Delete tasks when no longer needed.
* Clean and responsive dark-themed UI.
* Prevents adding empty tasks.

## 📋 How It Works

1. Enter a task in the input field.
2. Click the **Add** button.
3. The task appears in the task list.
4. Click **Done** to mark the task as completed.
5. Click **Delete** to remove the task from the list.

## 🎨 User Interface

* Dark-themed modern design
* Responsive layout
* Interactive buttons with hover effects
* Easy-to-read task cards
